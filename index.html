What is HTML?

HTML (HyperText Markup Language) is the standard language used to create documents on the World Wide Web. It acts as the skeleton of a webpage, defining the structure and content that users see.

How it Works

HTML uses "tags" to wrap content and tell the browser how to display it. These tags are enclosed in angle brackets. Most elements have an opening tag and a closing tag.

Example: <p>Text goes here</p>

The Core Structure

  • <!DOCTYPE html>: Tells the browser this is an HTML5 document.
  • <html>: The root element that contains everything else.
  • <head>: Holds meta-information like the page title.
  • <body>: Contains the visible content (text, images, links).
